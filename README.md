Chạy project trên WSL

### Cài đặt

#### 1. Tạo môi trường ảo

```bash
python -m venv venv
```

#### 2. Kích hoạt môi trường ảo

```bash
source venv/bin/activate
```

#### 3. Cài đặt thư viện

```bash
make install
```

#### 4. Chạy chương trình

```bash
make run
```

### Cấu hình

Cấu hình trong file:

```
config.py
```
